# 适用于微信、知乎、简书的markdown编辑器

##### 可用于hexo的独立网页

##### 示例网站：https://wangdudyb.gitee.io/wcmk/

## wcmk

### 优点

- 简约
- 布局比较美观

### 缺点

- 不带图片上传

## wd

### 优点

### 缺点

- 主题很少
